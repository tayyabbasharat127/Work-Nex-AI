# System Help

WorkNex includes dashboards for admin, manager, and employee roles. Dashboards use real backend APIs and show scoped workforce data.

Employees can check in, check out, view attendance history, apply for leave, inspect balances, and view performance. Managers can view direct team attendance, pending team leave, team performance, and subordinate lists. Admins can manage users, departments, settings, reports, notifications, and audit logs.

Power BI embedding requires backend environment variables for tenant, client, secret, workspace, report, and embed URL.
